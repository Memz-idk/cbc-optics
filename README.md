
CBC Optics
=
Adds sights to Create Big Cannons, compatible with Aeronautics.

Mainly adds the Cannon Sight item, attach it to the side of a big cannon and assemble the cannon, use either the Sight Viewer item or the 'Enter Sight' keybind to use it, leave it using the 'Leave Sight' keybind.

(Mod is still in beta so expect a couple bugs)
