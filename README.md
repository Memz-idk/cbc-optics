
CBC Optics
=
Adds sights to Create Big Cannons, compatible with Aeronautics.

Main Additions:
- Cannon Sight: Attach it to the side of a big cannon and assemble the cannon. To view the sight either use one of the keybinds or use a sight viewer, to leave use a keybind.
- Autocannon Sight: Attach it to an autocannon, also compatible with cannons from Advanced Technologies and Neo Warfare. Works very similarly to a normal sight.
- Optics Control Unit: Used to manage multiple sights. Use a sight linker item to link sights to the OCU, shift right-click with the sight linker to reset it, then use a sight viewer to attach the OCU to yourself, you can then use the keybinds to control it.

(Mod is still in beta so expect a couple bugs)
